# Orchardia Online V3.0.2 — Auto Stall Fix

แก้ปัญหาที่พบใน V2.x / V3.0.1:
เมื่อเปิด AUTO ไปสักพักหรือยืนนิ่ง จะมี Quest/Mini-map/เมนูด้านขวาโผล่แบบจาง ๆ และตัวละครหยุดชั่วคราวจน UI หาย

## สาเหตุ
Target สำหรับ UI และ Target ของระบบ AUTO ใช้ state เดียวกันมากเกินไป
เมื่อมอนสเตอร์ตาย/กำลัง Respawn/อยู่นอก Auto Leash:
- UI ยังคิดว่าอยู่ใน Combat
- แถบต่าง ๆ ถูกทำให้จาง
- AUTO อาจถือ Target ที่ไม่เหมาะสมหรือรอเป้าหมายเดิม

## แก้ไข
- แยกสถานะ `autoEngaged` ออกจาก UI Combat state
- UI จางเป็นเพียงเอฟเฟกต์ภาพ ไม่ควบคุม AUTO อีกต่อไป
- AUTO ตรวจ Target ทุกเฟรม
- Target ตาย/ออกนอก Leash จะถูกปล่อยทันที
- ถ้ามอนในจุดฟาร์มตายหมด AUTO จะค้นหากลุ่มใกล้เคียงภายใน 18 หน่วยแทนการยืนนิ่งรอ
- เมื่อพบกลุ่มใหม่ Auto Anchor จะย้ายตามอย่างปลอดภัย
- UI ที่จางไม่รับ Touch เพื่อไม่บังการควบคุม
- AUTO หลัง Respawn ยังทำงานต่อเหมือนเดิม
- คง Auto Loot / Auto Skill / Hide HUD / Open World / World Map

อัปทั้ง 4 ไฟล์แทน V3.0.1
