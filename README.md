# Orchardia Online V3.0 — Open World Foundation

ต่อยอดจาก V2.3 โดยเปลี่ยนสนามทดลองเดิมให้เป็นโลกขนาดใหญ่แบบเชื่อมต่อกัน

## โลก V3.0
1. Seedling Village / หมู่บ้านต้นกล้า — Lv.1–10
   - ตั๊กแตน
   - เพลี้ยฝูง
   - เมืองเริ่มต้น / ร้านค้า / NPC เดิม

2. Beetle Woods / ป่าด้วง — Lv.8–20
   - ด้วงเกราะ
   - หนอนกัดใบ
   - Boss: ราชาด้วงเกราะ

3. Sporeveil Marsh / บึงหมอกเห็ด — Lv.18–32
   - เห็ดสปอร์
   - ทากพิษ
   - แมลงวันบึง
   - Boss: ราชินีสปอร์

4. Sunscorch Desert / ทะเลทรายสุริยะ — Lv.30–45
   - แมงป่องทราย
   - หนอนทะเลทราย
   - Boss: ไททันหนอนทราย

5. Frostleaf Highlands / ที่ราบสูงใบเยือกแข็ง — Lv.42–60
   - ผีเสื้อน้ำแข็ง
   - โกเลมน้ำแข็ง
   - Boss: ผู้พิทักษ์เขาน้ำแข็ง

## ระบบใหม่
- ขยายขอบเขตโลกจากประมาณ ±14 เป็น ±47
- ถนนเชื่อม 5 ภูมิภาค
- ภูมิประเทศและ Landmark แยกตามโซน
- Monster Territory: มอนสเตอร์แต่ละชนิดเกิดตามพื้นที่ของตัวเอง
- Regional Boss 4 ตัว
- Boss respawn 60 วินาที
- Quest Tracker เปลี่ยนตามโซน
  - ฆ่ามอนสเตอร์ในพื้นที่ 5 ตัว
  - ต่อด้วยภารกิจปราบบอส
- Zone Banner แจ้งชื่อพื้นที่และระดับแนะนำเมื่อเดินข้ามโซน
- เก็บสถิติการฆ่ามอนสเตอร์/บอสแยกตามพื้นที่
- คง AUTO / Auto Loot / Auto Respawn / Hide HUD
- รองรับ iPhone / Android / Tablet / PC
- Flat Project 4 ไฟล์สำหรับ GitHub Mobile

## ขั้นต่อไป
V3.1 จะต่อยอด World Map, Fast Travel, NPC/Quest Chain และระบบโหลดโซนแบบ Chunk เพื่อให้ขยายโลกได้มากขึ้นโดยไม่ทำให้มือถือหนักเกินไป
