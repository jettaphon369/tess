# Orchardia Online V1.6 — Professional MMORPG HUD

ต่อยอดจาก V1.5.1 โดยคงระบบเดิมทั้งหมด

## UI ใหม่
- Status ตัวละครซ้ายบน: Level / HP / MP / EXP
- เงินและไอเทมขวาบน
- Mini-map
- Quest Tracker
- เมนูด้านขวา: กระเป๋า / ร้านค้า / เควสต์ / ล็อกเป้า
- Joystick ซ้ายล่าง
- Combat Cluster ขวาล่าง: โจมตี / Skill 1 / Skill 2 / กลิ้ง / คุย
- Cooldown แสดงกลางปุ่ม
- ปุ่มคุยแสดงเฉพาะเมื่อมีสิ่งให้โต้ตอบ
- กระเป๋า ร้านค้า เควสต์ และ Save ยังใช้ระบบเดิม
- รองรับ Safari / Chrome / iPhone / Android
- Flat Project 4 ไฟล์สำหรับ GitHub Mobile
