# Orchardia Online V3.1.1 — Frostleaf Visible Fix

แก้จาก V3.1 หลังทดสอบแล้วภาพยังดูใกล้เคียงเดิม

## แก้แบบเห็นชัด
- Frostleaf เปลี่ยนพื้นทั้งโซนเป็นหิมะสีขาวชัดเจน
- ถนนเหนือเปลี่ยนเป็นเส้นทางน้ำแข็ง
- เพิ่มทะเลสาบน้ำแข็งขนาดใหญ่
- เพิ่มป่าคริสตัลขนาดใหญ่
- เพิ่มประตูภูเขาน้ำแข็ง
- เพิ่มลานบอสวงกลมขนาดใหญ่
- เพิ่มปากถ้ำน้ำแข็งสีเข้ม
- กล้องยกสูงขึ้นเล็กน้อยเมื่ออยู่ Frostleaf เพื่อเห็นภูมิประเทศมากขึ้น
- ของตกแต่งหมู่บ้านเดิมจะไม่รบกวนภาพโซนไกลมากเท่าเดิม

## HUD
- Quest อยู่ช่องซ้ายของตัวเอง
- Target HP ย้ายลงมา ไม่ทับ Quest
- Zone Banner ย้ายลงและหายเร็วขึ้น
- คง Utility Rail แสดงตลอด
