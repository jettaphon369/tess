# Orchardia Online V2.3 — Auto Respawn & Hide Combat HUD

ต่อยอดจาก V2.2.1

## 1. AUTO ทำงานต่อหลังตายและเกิดใหม่
- เมื่อตัวละครตาย ระบบ AUTO จะไม่ถูกปิด
- หลัง Respawn จะล้างเป้าหมายเก่า
- ตั้งจุด Auto Anchor ใหม่ที่ตำแหน่งเกิด
- AUTO หาเป้าหมายใหม่และต่อสู้ต่อเอง
- Auto Skill และ Auto Loot ยังทำงานต่อ
- มีข้อความแจ้ง “เกิดใหม่แล้ว · AUTO ทำงานต่อ”

## 2. ซ่อนแถบปุ่มควบคุม
- เพิ่มปุ่ม “ซ่อนปุ่ม”
- ซ่อน Combat Cluster และ Joystick เพื่อดูเกมแบบสะอาดตอนฟาร์ม AUTO
- AUTO ยังคงต่อสู้ ใช้สกิล และเก็บของตามปกติ
- มีปุ่มลอย “⚔ แสดงปุ่ม” สำหรับเรียก HUD กลับ
- HUD สำคัญด้านบน เช่น HP/MP/EXP และ Target HP ยังแสดงอยู่
- PC กด H เพื่อซ่อน/แสดงปุ่มได้

## คงระบบเดิม
- Smart Auto
- Persistent Auto ขณะเปิดกระเป๋า/ร้านค้า/เมนู
- Auto Loot
- Auto Skill ON/OFF
- Responsive iPhone / Android / Tablet / PC
- Flat Project 4 ไฟล์
