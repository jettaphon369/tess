# Orchardia Online V1.7 — Combat & Character Polish

ต่อยอดจาก V1.6 โดยคงระบบ MMORPG HUD และระบบเดิมทั้งหมด

## เพิ่ม/แก้
- จัดชุดปุ่มต่อสู้ใหม่ให้ไม่เบียดกัน
- ย้าย Target Chip ไม่ให้ถูกปุ่มบัง
- เพิ่ม Target HP Bar ด้านบนเมื่อมีเป้าหมาย
- แสดงชื่อเป้าหมายและระยะห่าง
- Quest / Mini-map / Utility Menu จางลงขณะต่อสู้
- เพิ่ม Damage Number และ Critical Number
- เพิ่ม Hit Flash ศัตรู
- เพิ่ม Red Vignette เมื่อผู้เล่นโดนโจมตี
- เพิ่ม Procedural Animation ของตัวละครเวลาเดิน/โจมตี
- เพิ่ม Slash Effect
- เพิ่ม Enemy Bob / Hit Reaction
- Lock Target ช่วยจัดกล้องเข้าหาศัตรูมากขึ้น
- รองรับ Safari / Chrome / iPhone / Android
- Flat Project 4 ไฟล์สำหรับ GitHub Mobile
