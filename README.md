# Orchardia Online V2.2 — Auto Loot & Persistent Auto

ต่อยอดจาก V2.1

## แก้ตามที่ขอ
1. เมื่อ AUTO ON ของดรอปจะเข้ากระเป๋าอัตโนมัติภายในพื้นที่ Auto Leash
   - เหรียญ
   - ยา HP
   - เมล็ดหายาก
   ไม่ต้องเดินเหยียบของ

2. AUTO ไม่หยุดเมื่อเปิด UI
   - เปิดกระเป๋าได้
   - เปิดร้านค้าได้
   - เปิดเควสต์/บทสนทนา/เมนูได้
   - AUTO ยังต่อสู้และเก็บของต่อ
   - การเปิด UI จะหยุดเฉพาะการควบคุมด้วยมือ
   - ผู้เล่นตายจึงจะปิด AUTO ตามเดิม
