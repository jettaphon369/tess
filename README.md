# Orchardia Online V2.0 — Responsive Cross-Platform HUD

ต่อยอดจาก V1.9 และจัด HUD ให้ใกล้แบบตัวอย่างที่เลือก

## รองรับ
- iPhone Safari / Chrome
- Android Chrome
- Tablet แนวตั้ง/แนวนอน
- PC Browser

## Mobile
- Joystick
- Attack
- Skill 1
- Skill 2
- Roll
- Interact
- Lock Target
- AUTO
- Auto Skill ON/OFF

## PC
- WASD เดิน
- Space โจมตี
- 1 Skill 1
- 2 Skill 2
- Shift กลิ้ง
- F คุย
- R AUTO
- Tab Lock Target
- B กระเป๋า

## UI
- Target HP Bar แยกจาก Player HUD
- AUTO และ Auto Skill วางด้านขวา
- Combat Buttons เป็นวงโค้ง
- System Log สำหรับจอใหญ่
- Responsive สำหรับมือถือ/แท็บเล็ต/PC
- Safe Area สำหรับ iPhone
- ป้องกัน double-tap และ pinch zoom
- Flat Project 4 ไฟล์
