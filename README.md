- Zone Banner ย่อและหายเร็วขึ้น
- Quest Tracker ย่ออัตโนมัติระหว่างต่อสู้ แต่ไม่หาย
- Target HP แยกจาก Quest ชัดขึ้น
- World Map button เด่นขึ้น
- AUTO ใช้ระยะหยุดตามชนิดศัตรู

- ศัตรูมี soft separation ลดการกองทับกัน
- Frostleaf เพิ่มพื้นหิมะ, Ice Cave, Boss Arena, Frozen Cliff และ Crystal/Pine
