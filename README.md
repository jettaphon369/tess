# Orchardia Online V2.1 — Mobile Combat HUD

ต่อยอดจาก V2.0.1 Hotfix โดยไม่รื้อระบบเดิม

## ปรับ UI ตามแบบที่เลือก
- Combat Cluster แบบวงโค้ง
- Attack ใหญ่ขวาล่าง
- Skill 1 / Skill 2 กระจายรอบ Attack
- Dodge แยกชัดเจน
- AUTO / Auto Skill อยู่ด้านบนของกลุ่มปุ่ม ไม่ทับสกิล
- Lock Target เป็นปุ่มเล็กบริเวณชุดต่อสู้
- ปุ่มคุยซ่อนเมื่อไม่มี NPC/หีบ/ร้านค้า
- ปุ่มออกถูกย้ายเข้าเมนู ☰

## Responsive Profiles
- iPhone จอเล็ก
- iPhone/Android ปกติ
- Tablet
- PC
- Mobile Landscape

## เพิ่ม
- Low HP Warning ขอบจอแดงเมื่อ HP ≤ 25%
- ระยะ Auto Combat หยุดห่างจากศัตรูมากขึ้น ลดการซ้อนตัวละคร
- Safe Area รองรับ iPhone
- คง Smart Auto, Auto Skill, Target HP, Damage Number, Inventory, Shop, Quest, Save
