# Orchardia Online V3.1.2 — Combat Fixes

อัปเดตแก้ 3 จุดตามรีพอร์ตล่าสุด

## 1) ล็อกเป้า
- ย้ายปุ่ม `ล็อกเป้า` เข้าไปอยู่ในแถบสกิลด้านขวาล่าง
- ไม่ให้ลอยไปอยู่กลางจออีก
- กดแล้วจะติดสถานะ locked ชัดเจน

## 2) Quest / Target HP ไม่ทับกัน
- ขยับการ์ดเควสขึ้น
- ย่อขนาดการ์ดเควสเล็กน้อย
- เลื่อนแถบเลือดเป้าหมายลงมาให้อยู่คนละแถว
- เลื่อน zone banner ลงตามไปด้วย

## 3) มอนสเตอร์โจมตีแล้วดาเมจเข้า
- ลดแรงผลักชนของมอนสเตอร์
- ปรับระยะ melee ให้โจมตีได้จริง
- ตั้ง cooldown การโจมตีใหม่
- เวลามอนชนติดตัวจะทำดาเมจได้จริง ไม่ใช่แค่ชนแล้วดัน

คงระบบเดิมทั้งหมด:
- AUTO
- Auto Skill
- Auto Loot
- Auto Respawn
- Hide HUD
- Open World / Frostleaf
